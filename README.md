# Home Service Hero

Home Service Hero is a React + TypeScript web application (Vite) for booking home services and managing helpers/technicians. The UI uses Tailwind CSS and shadcn-ui components.

## Features

- Browse and request home services
- Helper/technician matching
- Booking flow with in-app communication
- Admin/management pages (where applicable)

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui
- Supabase (integration)

## Getting Started

### Prerequisites

- Node.js and npm

### Install

```sh
npm i
```

### Run in development

```sh
npm run dev
```

### Build

```sh
npm run build
```

### Preview build

```sh
npm run preview
```

### Lint

```sh
npm run lint
```

## Environment Variables

Check the Supabase and other integrations in `src/` for required env vars (commonly stored in `.env` during local development).
